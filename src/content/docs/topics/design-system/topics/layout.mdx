---
title: 布局
---

布局决定了模块或内容的许多属性：

+ 尺寸
+ 位置
+ 排列方式
+ 对齐方式
+ 堆叠方式（层级）

## 栅（shān）格

类似于栅（zhà）栏或条形码，将大块的区域分割为多个行或列，适用于平面设计和 UI 设计领域。

小区域的尺寸可以不相同，其间也可以有间隔。布局、模块和内容应当与小区域或间隔的边缘或中间对齐。

常见的方式是将页面的内容区域均分为 12/24 列。

## 网格

将页面均分为正方形网格，并将内容与其边缘或中间对齐。

常见的网格尺寸：

+ 4*4
+ 5*5
+ 8*8
+ 10*10
+ 16*16
+ 20*20

## 基本布局

相邻元素或父子元素之间的布局方式。

+ 线性布局（x/y，单一维度）：相邻元素横向或纵向依次排列
+ 网格布局（x + y，两个维度）：将父元素按行和列划分为多个子区域，子元素可以占据其中的一个区域，或横跨多个区域
+ 堆叠（z，一个维度）
+ 对齐：根据边缘、中线或中心点
+ 其它：
  + 滚动
  + 缩放
  + 瀑布流
  + 响应式

## 页面布局

+ 按照栅格进行布局
+ 划分为若干行和列，将主要区域留给内容
