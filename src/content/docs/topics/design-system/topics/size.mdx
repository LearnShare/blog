title: 尺寸

## 命名

+ 2xs: 特别小
+ xs: 更小
+ sm/s: 小
+ md/m: 中等（默认尺寸）
+ lg/l: 大
+ xl: 更大
+ 2xl: 特别大

## 使用场景

+ 页面布局（layout）
+ 模块间距（margin/gap）
+ 边框（border）
+ 圆角（border-radiuss）
+ 模块内边距（padding）
+ 文字（font-size/line-height/letter-spacing）

## 对齐网格

1. 线条：为了保证内容的显示清晰度，应当尽量使用整数（1x）尺寸，并将线条与物理（或显示）像素对齐
2. 文字：字号、行高应当使用偶数（2x）
3. 局部间距（包含边框）：使用偶数（2x）
4. 模块尺寸和间距：通常使用 4x/5x
