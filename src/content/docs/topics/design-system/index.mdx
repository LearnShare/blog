## 操作指南

1. 调研和总结相关资料
2. 明确基本原则及目标
3. 划分内容模块，逐步细化
  1. 参考相关资料
  2. 明确模块目标
  3. 明确规则，抽象为参数和数据
  4. 试用和调整
4. 参考成型的规范进行案例设计，查漏补缺
5. 实现 CSS 库
6. 实现 Web 组件库
